## Getting started

Requires [NodeJS](https://nodejs.org/en/) and [Yarn](https://yarnpkg.com/en/).

```bash
# Clone the repository
git clone https://github.com/amercier/javascript-exercise.git
cd javascript-exercise

# Install dependencies
yarn

# Run tests
yarn test
```
